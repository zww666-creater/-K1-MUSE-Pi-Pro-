基于进迭时空K1 MUSE Pi Pro 的老年多模态生活辅助与数字引导系统
摘要	
在我国已经进入老龄化社会的大背景下，本作品面向老年用户在智能手机操作、纸质信息阅读和生活场景信息获取中的实际困难，基于进迭时空 K1 MUSE Pi Pro 边缘计算平台，设计了一套老年多模态生活辅助与数字引导系统。系统以“按键交互 + 视觉理解 + 执行反馈”为核心思路，采用本地化部署方式，将摄像头采集、微信页面识别、语音播报、按键控制和 OCR 拍照朗读等功能整合到同一嵌入式平台中，形成低门槛、可解释、可扩展的数字助老原型。在视觉感知方面，系统通过摄像头采集手机界面图像，使用部署在本地的 ONNX 图像分类模型识别微信首页、聊天页面、功能菜单和视频通话页面，并通过状态机与重复播报控制机制向用户输出页面提示，从而辅助老年人完成微信聊天、功能选择和视频通话等常见操作。在交互方面，系统接入低电平触发的 4P 按键模块，将 1 号键定义为 OCR 识别入口，2 号键定义为微信功能引导入口，3 号键定义为项目启动入口，4号按键定义为退出当前模式，进入待机状态，降低了触屏操作和语音唤醒对环境噪声、口音和误触的依赖。在语音反馈方面，系统采用缓存 wav 播放机制代替实时 TTS 合成，减少了语音输出等待时间，提高了离线环境下的稳定性。工程实现上，当前版本已完成 ONNX Runtime CPU 推理、微信页面分类模型加载、缓存语音播报、GPIO 按键监听、启动器拉起主程序和基本运行脚本整理；OCR 拍照识别流程已完成按键入口设计，后续需要进一步联调“拍照、识别、朗读”的完整闭环。测试结果表明，在 CPU Provider 环境下，将 ONNX Runtime 配置为 8 个 intra-op 线程后，模型平均推理时间由单线程约 2677.93 ms 降至约 454.56 ms，能够满足页面级智能引导的交互需求。作品具有本地运行、隐私友好、按键操作直观、模块可扩展和场景适配性强等特点，可应用于居家养老、社区助老、适老化智能终端和弱视人群辅助阅读等场景。
关键词：K1 MUSE Pi Pro；边缘AI；适老化交互；微信页面识别；OCR；语音播报；按键控制
